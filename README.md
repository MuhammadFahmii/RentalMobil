# RentalMobil
Simple implementation MVC DAO Java

### Build With

- Java
- Library JDateChooser
  
### Installation

1. Import database ke Webserver
2. Jalankan Webserver
3. Jalankan dengan klik Run pada IDE (Netbeans) atau click f6
4. Enjoy
